## 单步调试/跟踪

有时可能跟踪程序的运行也能帮助对程序的理解，启动单步调试的方法是：

在 vscode 里打开该项目，然后在单元测试文件 [test-all.js](./test/test-all.js) 或者源码 [aho-corasick.js](./src/aho-corasick.js) 里设置断点，然后执行 `Run and Debug` 即可。
